What is Web Development?


Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites.
Web Development can be classified into two
1. Frontend Development
2. Backend Development

DataBase Type

1. Front-end Web Developer
A front-end web developer is a person who designs web through coding languages like HTML, CSS and JavaScript.
A Front-end web developer is also called a web designer.
That is, the front-end web developer designs the cable client-side, such as HTML documents across different
devices, and controls the screen size, etc.
The Front-end web developer user interface simplifies the experience.
A front-end web developer is responsible for making the web page easy for the website visitor, making it easier
for the visitor to interact with the website.

2. Backend Web Developer
“ Backend ” refers to any part of a website or software program that is not visible to the user. A backend web
developer is a person who specializes in programming languages ​​such as PHP, Java, Ruby, Python, and .Net and
does server-side development.
A backend web developer specializes in:
Backend Web Development Languages: PHP , Java , .Net , Python etc.
Database and Cache: MySQL , MongoDB , Oracle , and SQLServer etc. and has knowledge of caching mechanisms
Memcached etc.
Server: Handles Apache, Nginx, Microsoft IIS Server etc.
API: Its full name is "Application Programming Interface", which helps the client and server to communicate with
each other.

3. Database Type
A database is an organized collection of structured information, or data, typically stored electronically in a
computer system. A database is usually controlled by a database management system (DBMS). Together, the data and
the DBMS, along with the applications that are associated with them, are referred to as a database system, often
shortened to just database.
Data within the most common types of databases in operation today is typically modeled in rows and columns in a
series of tables to make processing and data querying efficient. The data can then be easily accessed, managed,
modified, updated,controlled, and organized. Most databases use structured query language (SQL) for writing and
querying data.
